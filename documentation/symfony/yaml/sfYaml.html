		static load (line 67)
	
 
	
	

Loads YAML into a PHP array.


The load method, when supplied with a YAML stream (string or file),  will do its best to convert YAML in a file into a PHP array.
Usage:   
  1.    $array = sfYaml::load('config.yml');
    2. 

  2.    print_r($array);
    3.
